# Workshop IPv6 e OSPFv3
**Lab feito no EVE-NG do Workshop de IPv6 e OSPFv3 ministrado pelo Gustavo Kalau**

<br>

**Passos realizados:**
1. Habilitação do IPv6 nos roteadores e interfaces - Endereço Link-Local
2. Configuração do endereço Global (GUA) nas interfaces LAN dos Roteadores R1 e R5 - e Verificação do SLAAC
3. Configuração do OSPFv3 nos Roteadores
4. Configuração das loopbacks dos Roteadores com Unique local (Começa com FC..)
5. Configuração do IPv4 no OSPFv3
